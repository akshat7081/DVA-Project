# Social Media Trends Analytics Dashboard

**DVA Assignment 2 | Akshat Tripathi | 04914202023 | IPU BCA 6th Semester**

## Project Overview
A multi-page interactive dashboard built with Python, Tkinter, Pandas, and Matplotlib to analyze social media trends and engagement metrics across Twitter, Instagram, YouTube, Facebook, and TikTok.

## Features
- **Overview Page** — KPI cards, platform reach, category distribution, top hashtags
- **Platform Analysis** — Detailed table + bar charts comparing all 5 platforms
- **Category Trends** — Content category performance with engagement metrics
- **Hashtag Rankings** — Ranked table + top/bottom 10 visualizations
- **Heatmap & Correlation** — Metric correlation matrix + Platform×Category engagement heatmap

## Tech Stack
| Technology | Purpose |
|-----------|---------|
| Python | Programming language |
| Tkinter | GUI framework |
| Pandas | Data analysis |
| Matplotlib | Visualization |
| NumPy | Numerical computation |

## Dataset
Custom generated dataset with 2000 records containing:
- Platform, Hashtag, Category, Country
- Posts, Likes, Views, Comments, Shares
- Engagement Rate (%)
